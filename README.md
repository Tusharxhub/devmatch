

# 🚀 DevMatch 

A sleek and developer-friendly frontend for **DevMatch** – a platform to connect developers for hackathons, collaborations, and remote jobs, powered by GitHub data and modern UI/UX.

> 🛠 Built by [Tushar Kanti Dey](https://github.com/Tusharxhub)
> 🎨 UI only – Backend is managed separately

---

## 🌟 Features

* 🔐 GitHub OAuth login UI (integration-ready)
* 💻 Responsive developer profiles with GitHub data
* ❤️ Swipe-based "Like/Dislike" interaction
* 💬 Chat interface mockup (Push Protocol ready)
* 📨 Match notification modal design
* 🧠 Remote job suggestion section UI

---

## 🛠️ Tech Stack

* **Framework**: React.js
* **Styling**: Tailwind CSS
* **Routing**: React Router / Next.js (depending on your setup)
* **Animation**: Framer Motion (if used)
* **Forms/Modals**: Headless UI / Custom components

## 📦 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Tusharxhub/devmatch.git
cd devmatch
```

### 2. Install Dependencies

```bash
npm install
# or
yarn install
```

### 3. Run the Development Server

```bash
npm run dev
# or
yarn dev
```

Frontend should be available at `http://localhost:3000`
